Ng-Set-State it is a library that helps developing angular components in a more functional style where UI logic is representing as a series of immutable state transitions.

  • Angular Components with Extracted Immutable State - an article on Medium that explains the ideas of this library.

State internal dependency graph

Save request takes 3 seconds and sometimes it fails (this behavior is simulated). However, you can edit “TODO” items in real time – all changes will be queued and will be performed when previous operations are done. If you leave the tab when some saving operation is running the list will be destroyed, but operation will be continued, and state will be updated. It can lead to a component blocking to prevent states conflict.

Also, the list waits for a 2 second period of "silence" when there are no any changes and only then it starts saving the changed data.
